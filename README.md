### 关键特性

* 64MHz Cortex-M0
* 36KB SRAM + 64/128KB Flash
* 2.5 Msps 12-bit ADC
* 4x USART 8Mbps
* USB Type-C供电控制器
* HDMI CEC接口
* 1.7 to 3.6 V
